Cinema E-ticket

1st and 2nd tasks of Russian Design Cup 2012

Tasks: 1. Make a card payment page. 2. Make a printed cinema e-ticket.

The payment should be quick and easy so the user could spend less time on it. That's why it's designed in details.

After purchasing you can print a ticket. In this particular task used doesn’t have to additionally exchange e-ticket to a common cinema ticket at the ticket-office. So he can move straight to the movie. All the booked seats come with the same ticket. As far as user has to print it we can at least put some proper information on it: cinema location, bar prices, etc.

The printed version of the ticked should have good contrast to be okay on any printer and to have no problems with scanning the qr-code. For the same reason there is a minimum of graphics and colors there.
